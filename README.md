During this project, I focused on creating a clean, side-by-side card layout featuring a gradient result panel on the left and a detailed dynamic-looking summary list on the right. 

One specific challenge I overcame was handling the document flow so that the attribution footer stayed neatly anchored at the bottom of the screen rather than floating awkwardly next to the component.
